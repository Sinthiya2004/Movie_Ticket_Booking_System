# Movie_Ticket_Booking_System
Developed a Movie Ticket Booking System using MySQL to manage movies, theatres, shows, seat reservations, ticket bookings, payments, snack orders, customer reviews, and discounts. Implemented SQL concepts including DDL, DML, Joins, Subqueries, Aggregate Functions, and Views for efficient data management and reporting.

Project Overview

The Movie Ticket Booking System is a MySQL-based database project designed to manage movie ticket reservations efficiently. The system handles movies, theatres, screens, shows, customers, bookings, tickets, payments, snack orders, reviews, discounts, and employee details.

Features

Movie and Theatre Management
Show Scheduling
Seat Reservation System
Ticket Booking and Payment Tracking
Snack Ordering Management
Customer Reviews and Ratings
